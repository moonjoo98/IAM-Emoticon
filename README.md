# IAM-Emoticon
내가 이모티콘이 된다면? StyleGAN2를 활용한 이모티콘 생성

### 배경
오늘날, 남녀노소 가리지 않고 연락을 주고 받을 때, 이모티콘을 사용한다. 하지만 상황에 맞는 이모티콘과 원하는 이모티콘을 사용하기 위해선 이모티콘을 구매해야한다.
